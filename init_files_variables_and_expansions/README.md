This project teaches the basics of shell behavior, starting with what happens when a command like ls -l *.txt is typed, how shell initialization files such as /etc/profile, /etc/profile.d, and ~/.bashrc work, and how variables function, including local vs global variables, reserved variables like HOME, PATH, and PS1, special parameters such as $?, and shell expansions, quotes, command substitution, arithmetic, aliases, and running commands from a file in the current shell. 
It also focuses on practical scripting tasks such as printing the current user, modifying PATH, counting directories in PATH, displaying environment and local variables, creating local and global variables, and performing arithmetic operations using values stored in environment variables. 
Finally, the project includes more advanced shell exercises like base conversions, floating-point formatting, generating combinations, and understanding how to expand and manipulate values correctly in Bash.
